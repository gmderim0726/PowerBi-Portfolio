# Credit Risk Analysis & Operational Monitoring

## The problem

A bank needed a clean visualization about the credit usage for counter the default risk and fight against the costumer churn. The projec goal was build an ent-to-end pipelina that ingest raw data to create management and operational dashboards.

## Key questions explored

1. How much is the total values of credit limit, revolving balance and churn rate?
2. Who has the biggest credit limit?
3. The churn is at control?
4. The card category affects the usage?
5. Who is closest do default risk?
6. Who is the profitable target audience?
7. The customers are satisfied?

## Getting started

To get start with Credit Analysis, you can follow these steps:

1. Open this repository on your machine.
2. With the Pyton pipeline, insert your PostgreSQL server information.
3. Load in the dataset on PostgreSQL.
4. Copy and paste the queries on PostgreSQL for create the Views.
5. Open your Power BI to interact with visualization and answer de Key questions.
